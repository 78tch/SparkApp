# SparkApp
## 1.画面ごとの機能
1. エフェクト
2. Smart Jam
3. オートコード

## 2.エフェクトの並び順
エフェクターの並び順は固定で、順番は入れ替えられない。
1. GATE
2. COMP/WAH
3. DRIVE
4. AMP
5. MOD/EQ
6. DELAY
7. REVERB
